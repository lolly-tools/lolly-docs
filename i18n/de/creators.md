# Lolly für Ersteller

Sie möchten schnell etwas Gutes erstellen, und zwar richtig. Genau dafür ist Lolly gemacht: Füllen Sie ein paar Felder aus und erhalten Sie eine fertige Datei - Layout, Typografie und Regeln werden vom Tool durchgesetzt, nicht von Ihrem Auge. Keine Design-Software, kein Warten. Es läuft auf Ihrem eigenen Gerät und funktioniert offline.

**Ihr Vorteil:** Sie liefern in Minuten, und es sieht aus, als hätten Sie Stunden investiert. Sie warten nie auf eine Designerin oder einen Designer, und weil die Markenregeln fest eingebaut sind, steht nie Ihr Name auf einem markenfremden Asset. Sie wirken einfach schnell, professionell und zuverlässig.

Neu hier? Beginnen Sie mit dem **[Schnellstart](/info/quickstart.html)** - er bringt Ihre Marke hinein und Ihre erste Datei heraus. Diese Seite zeigt Ihnen, warum Sie bleiben werden.

## Warum Lolly - Ihre Vorteile

- **Fertig, nicht fummelig.** Das Tool kennt bereits die Schriften, die Farben, das Logo, die Abstände. Sie liefern die Worte; es liefert das fertige Asset zurück. Nichts, worüber Sie nachdenken müssen.
- **Von Natur aus korrekt.** Designregeln, Layout-Vorgaben und Ablauflogik stecken in der Vorlage - das Ergebnis kann nicht von der Spezifikation abweichen, es sei denn, die Autorin oder der Autor des Tools lässt bewusst etwas offen. Wilde kreative Bandbreite, innerhalb von Leitplanken, die halten.
- **Eine Quelle, jedes Format.** Dasselbe Design lässt sich als SVG, druckfertiges PDF, hochauflösendes PNG oder 60-fps-Video exportieren. Einmal erstellen, überall ausliefern.
- **Unbegrenzte Variationen.** Lokalisieren Sie eine Karte in ein Dutzend Sprachen, erzeugen Sie jede Größe für jeden Platzierungsort, tauschen Sie ein Lockup in vierzig Kampagnen aus - alles aus einem einzigen Design, gesteuert von einer Tabelle.
- **Es gehört Ihnen, offline, privat.** Es wird nichts hochgeladen. Kein Konto, keine Cloud, keine Lizenz, die geprüft werden muss. Schließen Sie im Flugzeug einfach den Laptop und arbeiten Sie weiter.
- **Offen, kein Lock-in.** Die Engine ist Open Source, und Ihre Arbeiten liegen unter einfachen URLs, nicht in der Datenbank eines Anbieters.

## Was Sie erstellen können

### Alles korrekt, ohne Wartezeit

Füllen Sie ein paar Felder aus und erhalten Sie das fertige Asset - keine Schriften auswählen, keine Farben infrage stellen, kein „Ist das das richtige Logo?" Das Tool weiß es bereits. E-Mail-Signaturen, Event-Countdowns, Zitatkarten, Badges, Straßenkarten - alles, was Ihr Team an einem Dienstag um 16 Uhr braucht, im Selbstbedienungsprinzip. Und weil keine Kundendaten das Gerät verlassen, entsteht bei der Zusammenarbeit kein Compliance-Risiko.

### Von Live-Daten zum fertigen Visual

Verbinden Sie eine **CSV**- oder **JSON**-Tabelle mit einer Diagramm-, Karten- oder Tabellenvorlage und aktualisieren Sie die Zahlen - das Layout kümmert sich von selbst darum. Wahlergebnisse, Quartalszahlen, Wetterereignisse: Erstellen Sie das Format einmal und nutzen Sie es jedes Mal wieder, wenn die Story läuft. Treffen Sie Ihren Corporate-Style *exakt*, denn er wird auf Vorlagenebene durchgesetzt, nicht als lose Richtlinie belassen.

### Tools, die andere Tools einbinden

Ein Tool kann die Live-Ausgabe eines anderen Tools als Asset einbetten - ein Namensschild, das seinen eigenen QR-Code rendert, eine Karte, die ein Live-Diagramm einfügt - ganz ohne Design-Tool und ohne manuelles Compositing. Das eingebettete Render bleibt live und wird beim Laden neu gerendert; es ist nie ein flach gerendertes Bild.

### Die Grenze des Machbaren, wenn Sie sie brauchen

Fortgeschrittene Filter, Compositing, Animation, Live-Kameraeffekte, generatives Design - jede Webtechnologie steht einer Vorlagen-Autorin oder einem Vorlagen-Autor zur Verfügung, die Obergrenze liegt also hoch. Erstellen Sie *Tools*, keine Dateien: Bauen Sie das, was 10.000 Karten generiert, statt 10.000 Karten von Hand zu erstellen. Sperren Sie, was sich nicht ändern soll; geben Sie nur frei, was variieren soll.

## Details, die Ihnen auffallen werden

- **Druck, richtig gemacht.** Echte physische Einheiten (mm/cm/in/pt) und DPI, Beschnitt und Schnittmarken, CMYK und PDF/X-4-Output-Intent - eine Druckerei bekommt genau das, was sie braucht. Siehe [Export](/info/exporting.html#size-print-units).
- **Video und Bewegtbild.** Export als WebM/MP4 (mit Audiospuren), animiertes PNG/WebP, animiertes SVG und Lottie - aus derselben Vorlage, die auch Ihre Standbilder erzeugt.
- **Editierbare Vektoren.** Export als SVG, EPS, EMF und DXF. Im Vektor-Export wird Text zu umrissenen Pfaden, sodass eine Druckerei oder ein Cutter-Gerät scharfe Geometrie erhält, die sich überall öffnen lässt. (Umrissener Text ist in Illustrator nicht mehr *neu editierbar* - das ist der Kompromiss für perfekte Treue ohne die Schriftdatei.)
- **Komfort, der sich summiert.** Verarbeiten Sie eine Tabelle im Batch zu einem Ordner voller Assets; speichern Sie Sitzungen und organisieren Sie sie in Projekten; teilen Sie jeden Zustand als URL; rendern Sie einen ganzen Ordner als eine ZIP-Datei.
- **Eine UI, die tatsächlich angenehm ist.** Undo/Redo überall, eine Sound-Ebene, die sich abschalten lässt, ein Theme-Wechsel, Live-Kamera-Filter, eine Masonry-Galerie - und der **Neurospicy Mode** für eine ruhigere, reizärmere Oberfläche.
