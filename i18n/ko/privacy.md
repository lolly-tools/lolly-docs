# 개인정보 처리방침

*최종 업데이트: 2026년 6월*

## Lolly 앱

Lolly는 전적으로 브라우저 안에서 동작해요. **저희는 아무것도 수집하지 않고, 아무것도 전송하지 않으며, 사용자의 데이터를 볼 수 있는 서버가 전혀 없어요.** 어떤 분석 도구도, 추적도, 제3자도 존재하지 않아요.

**어디에도 쿠키가 없어요.** Lolly는 쿠키를 설정하는 일이 전혀 없어요. 앱이 동작하도록 **사용자 자신의 기기에** 소량의 데이터를 보관하는데, 이는 모두 사용 중인 기능에 반드시 필요한 것들이에요.

- **라이트/다크 테마**와 몇 가지 인터페이스 설정(사이드바 너비, 확대/축소).
- **도구 카탈로그의 오프라인 캐시** — 연결이 없어도 갤러리가 계속 로드되도록 해요.
- 프로필 카드의 소소한 통계를 위한 **로컬 전용 사용량 카운터** — 이 값은 어디로도 전송되지 않아요.
- **사용자 자신의 문서와 저장된 세션** — 브라우저에 로컬로(IndexedDB) 저장되어 방문할 때마다 작업 내용이 유지돼요.

이 중 어느 것도 공유되거나 업로드되지 않고, 사용자를 식별하거나 추적하는 데 쓰이지도 않아요. 그러니 동의를 받아야 할 것은 없고, 무엇이 보관되는지 알려드리기 위한 이 안내문만 있을 뿐이에요. **프로필 → 내 데이터 전체 삭제**를 사용하거나 브라우저에서 사이트 저장소를 지우면 언제든 전부 없앨 수 있어요.

이 문서 사이트(`/info`)는 한층 더 가벼워요. **쿠키를 전혀 설정하지 않고**, 기기에는 라이트/다크 설정만 저장하며, 폰트를 포함한 모든 것을 lolly.tools 자체에서 제공해요 — CDN이나 제3자 요청은 전혀 없어요.

## 온디바이스 유틸리티

일부 도구는 *사용자 본인이* 제공한 파일을 다루는 **유틸리티**예요 — 예를 들어 이미지나 PDF에 숨어 있는 데이터(GPS 위치, 카메라, 작성자, 편집자, 문서 메타데이터)를 보여주고 깨끗한 사본을 돌려주는 **Strip Hidden Data**, 또는 기기에서 바로 이미지를 재인코딩해 PDF 용량을 줄이는 **Compress PDF**가 있어요.

이 도구들은 **전적으로 브라우저 안에서** 실행돼요. 선택한 파일은 기기의 메모리로 읽어 들여 로컬에서 변환한 뒤 다운로드로 제공해요. **업로드되는 일은 절대 없어요** — 애초에 업로드할 서버가 없기 때문이에요. 정리된 사본에는 워터마크도, 저희 자체 식별용 메타데이터도 전혀 없어요. 핵심은 데이터를 *추가*하는 게 아니라 *제거*하는 것이니까요. 페이지를 떠난 뒤에는 아무것도 저장되지 않고, 이 유틸리티들은 오프라인에서도 동작해요. 이 도구들 모두에서 **"내 기기에서 실행——아무것도 업로드되지 않음"** 배지를 보게 될 거예요.

이것은 브라우저가 로컬에서 처리할 수 있는 작업을 위해 파일을 낯선 서버로 업로드하는, 전형적인 "이 PDF 압축하기" / "이 HEIC 변환하기" 웹사이트와는 정반대예요.

## 브라우저 확장 프로그램

**Lolly URL Screenshot** 브라우저 확장 프로그램은 어떤 개인정보도 수집·저장·전송하지 않아요. 분석 도구도, 추적도, 원격 서버도 없어요.

## 동작 방식

Lolly 웹 앱([lolly.tools](https://lolly.tools))에 URL 스크린샷을 요청하면, 확장 프로그램이 임시 백그라운드 탭에서 해당 페이지를 열고 DevTools Protocol을 사용해 브라우저 안에서 캡처한 뒤 이미지를 앱으로 돌려주고 탭을 닫아요. 모든 과정은 사용자 자신의 기기와 네트워크 안에서 로컬로 이뤄져요.

## 데이터

- **저희는 아무것도 수집하지 않아요.** 이 확장 프로그램에는 서버가 없고, 자체적으로 네트워크 요청을 하지도 않아요.
- **캡처된 이미지**는 같은 브라우저 안의 Lolly 앱으로 곧바로 전달돼요 — 확장 프로그램이 업로드하는 일은 전혀 없어요.
- **캡처하는 URL**은 오직 그 한 번의 스크린샷을 위해 해당 페이지를 불러오는 데만 사용돼요. 기록되거나 공유되지 않아요.

## 권한

- **`debugger`** — DevTools Protocol을 통해 렌더링된 페이지를 캡처하기 위해서예요(Lolly 데스크톱 앱이 사용하는 것과 동일한 방식이에요).
- **탭 접근** — 페이지가 로드되는 임시 탭을 열고 닫기 위해서예요.
- **호스트 접근** — 캡처 대상으로 선택한 페이지가 어떤 사이트에든 있을 수 있기 때문이에요.

이 권한들은 사용자의 브라우징 내용을 읽거나, 감시하거나, 전송하는 데 사용되지 않아요.
