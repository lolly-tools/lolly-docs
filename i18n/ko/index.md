
# Lolly — 랜딩 페이지 카피

프로덕션 수준의 파일을 몇 초 만에——디자인 규칙, 레이아웃 로직, 내보내기 품질이 이미 내장되어 있어요. 기다릴 필요도, 어림짐작할 필요도 없어요.
사용자의 기기에서 직접 실행돼요. 업로드할 것도, 배울 것도 없고, 오프라인에서도 동작해요.

---

## 마케터

### 기다림 없이, 모든 것이 완성돼요.

인용 카드, 이벤트 타일, 현지화된 서명이 필요한 건 다음 스프린트가 아니라 바로 오늘이에요. 디자인 도구를 한 번도 열어본 적이 없어도 Lolly는 완성되고 브랜드에 맞는 결과물을 돌려줘요. 규칙이 템플릿 안에 있기 때문에 결과물은 늘 올바르게 나와요.

- **몇 가지 항목만 채우면 완성된 에셋을 받아요.** 고를 폰트도, 고민할 색상도, "이게 맞는 로고인가?" 같은 의문도 없어요. 도구가 이미 답을 알고 있어요. 여러분은 글만 준비하면 돼요.
- **무한한 조합, 그러나 단일한 원천 정보.** 인용 카드를 12개 언어로 현지화하거나, 브랜드 락업을 40개 캠페인에 걸쳐 바꾸거나, 모든 배치용으로 모든 사이즈 버전을 생성하는 일——디자인을 손대지 않고도 할 수 있어요. 배치 그리드의 스프레드시트로 전부 구동하세요. CSV를 붙여넣거나 가져오면 행마다 완성된 에셋 하나씩 얻어요.
- **일상적인 에셋 제작에 크리에이티브 에이전시 병목이 없어요.** 이메일 서명, 이벤트 카운트다운, 인용 카드, 거리 지도——화요일 오후 4시에 팀에게 필요한 모든 것을 셀프서비스로 처리해요.
- **컴플라이언스 리스크 없이 협업해요.** 고객 데이터가 서드파티 클라우드 서비스로 업로드되지 않아요. 감사해야 할 SaaS 라이선스도 없어요. 모든 것이 로컬 기기 연산만으로 동작해요.

---

## 영업

### 어떤 미팅에도 필요한 것을 정확히 들고 들어가요.

이동 중인데 자료가 잘못됐고, 고객이 특정한 요청을 했어요. Lolly는 어떤 기기든 에셋 스튜디오로 바꿔줘요——디자이너도, 기다림도, 변명도 필요 없어요.

- **아무것도 기다릴 필요 없어요.** 이동 중이고 미팅까지 남은 시간이 없어도, 지금 당장 에셋이 필요하면 바로 생성하세요.
- **1인 군단.** 손바닥 안에서 발휘되는, 잘 조율된 글로벌 캠페인의 힘이에요. 모든 담당자, 모든 지역이 동일한 완성도를 갖춰요.
- **월드클래스 경험.** 즉시 렌더링됐어도 어느 것 하나 서두른 티가 나지 않아요. 일회성인 것은 없어요——그래서 상호작용을 거듭할수록 조금씩 더 다듬어져요.
- **팀의 속도에 놀랄 거예요!** 그들은 이미 잠들어 있어요. 제가 처음 필요했을 때 이미 해결돼 있었거든요.

## 저널리스트

### 속도와 정확성을 위해 만들어진, 여러분의 편집 툴킷.

편집 스타일을 한 번 구축해 두면, 실시간 데이터로부터 출판 품질의 에셋을 즉시 생성할 수 있어요.

- **데이터를 곧바로 비주얼로 가져와요.** 구조화된 데이터를 차트, 지도, 표 템플릿에 연결하세요. 숫자를 업데이트하면 레이아웃은 알아서 처리돼요.
- **여러분 매체의 스타일에 정확히 맞아요.** 도구는 타이포그래피와 색상 제약이 하드코딩된 상태로 제작돼요——하우스 스타일이 느슨한 가이드라인이 아니라 템플릿 단계에서 강제돼요.
- **인쇄용으로도, 화면용으로도.** 같은 템플릿에서 SVG, CMYK PDF, 고해상도 PNG, 60fps 동영상을 내보내요. 하나의 소스로 제작 데스크에 필요한 모든 포맷을 충족해요.
- **반복되는 기사를 위한 재사용 가능한 포맷.** 선거 결과, 분기 실적, 기상 이벤트——템플릿을 한 번 만들면 그 기사가 다시 나올 때마다 재사용해요.

## 운영

### 일상적인 산출물에 산업 수준의 엄격함을.

전부 마케팅은 아니에요. 탁송 라벨, 배지 대량 발급, 컴플라이언스 한 페이지 자료, 사고 통지——레이아웃 자체가 구조를 지탱하고 데이터가 절대 틀려서는 안 되는 산출물이에요. Lolly는 구조화된 콘텐츠를 최고 수준의 디자인과 동일한 엄격함으로 다뤄요.

- **구조화된 데이터를 넣으면, 정확한 출력이 나와요.** CSV나 JSON 표를 넣으면 모든 필드가 템플릿이 기대하는 자리에 정확히 놓여요——행마다 완성된 파일 하나씩이에요.
- **물리적 정밀도.** 실제 단위(mm/cm/in/pt), 정확한 DPI, 재단 여백과 재단선까지요. 라벨 프린터나 인쇄소는 필요한 것을 정확히 그대로 받아요.
- **로직은 도구 내부에서 실행돼요.** 바코드, 날짜 연산, 조건부 레이아웃, 명암 대비 검사——렌더링 시점에 계산되고, 수작업이 아니에요.
- **설계상 결정론적이에요.** 같은 입력이면 늘 같은 파일이 나와요. CLI나 파이프라인에서 감사 가능하고, 재현 가능하고, 자동화할 수 있어요.

## 미디어 & 크리에이티브

### 기계가 가장 잘하는 일은 기계에게 맡기세요.

여러분이 설계하는 건 시스템이고 해결하는 건 문제예요, 일회성 작업이 아니고요. Lolly는 여러분의 디자인 결정을 조직 전체가 쓸 수 있는 도구로 바꿔주는 실행 레이어예요——모든 에셋마다 여러분이 직접 관여할 필요가 없어요.

- **이것이 프런티어예요.** 고급 필터, 합성, 애니메이션, 생성형 디자인——이 플랫폼에는 천장이 없어요. 템플릿 작성자에게는 모든 웹 기술이 열려 있어요. 어떤 크리에이티브 스위트보다 더 많은 것을, 완성된 프로덕션 품질을 낼 수 있는 힘과 함께 해낼 수 있어요.
- **도구가 도구를 조합해요.** 한 도구가 다른 도구의 출력을 라이브 에셋으로 임베드할 수 있어요——자신의 QR 코드를 렌더링하는 명찰, 라이브 차트를 끼워 넣는 카드——디자인 도구도, 수작업 합성도 필요 없어요.
- **파일이 아니라 도구를 만드세요.** 소셜 카드 10,000장을 직접 만드는 대신, 10,000장을 생성하는 템플릿을 만드세요. 여러분의 시간은 디자인 문제에 쓰이고, 대량 생산 작업에는 쓰이지 않아요.
- **지루한 작업을 없애요.** 최적의 조판, 로고 배치, 지도 핀 위치 조정, 색상 대비 검사, 내보내기 사이즈 조절——모두 템플릿 레이어에서 해결돼요. 규칙은 한 번만 정의하고, 반복 작업은 기계에게 맡기세요.
- **바뀌면 안 되는 것은 고정하고, 바뀌어도 되는 것은 자유롭게 하세요.** 브랜드 제약은 하드코딩하세요. 실제로 달라져야 하는 변수만 노출하세요. 도구가 크리에이티브 가드레일이 돼요.

## 개발자

### 빌드 파이프라인에는 여러분의 비주얼도 포함되어야 해요.

이미지는 빌드 아티팩트예요. 그렇게 취급하세요. Lolly는 CLI에서 실행되기 때문에 다른 모든 것을 생성하는 방식 그대로 에셋을 생성할 수 있어요——반복 가능하게, 자동으로, 워크플로우의 일부로요.

- **생성형 AI 미디어 복권은 건너뛰세요.** 생성형 AI 미디어를 시도해 봤을 거예요——비싸고, 충분히 만족스럽지도 않았죠. Lolly는 무료이고 더 빠른, 로컬 콘텐츠 자동화로 AI 특유의 조악함과 여러분의 수고를 함께 없애줘요.
- **재현 가능한 출력.** 같은 입력은 늘 같은 파일을 만들어요. URL을 커밋하고 필요할 때 다시 생성하세요——더 이상 이미지를 커밋하거나 Slack에서 최신 버전을 찾아다닐 필요 없어요.
- **미디어를 코드베이스 밖으로 꺼내세요.** OG 이미지, QR 코드, 소셜 카드, 데이터 비주얼을 Git에 바이너리로 저장하는 대신 빌드 시점에 생성하세요.
- **에셋 안에서 로직을 실행하세요.** Code Canvas나 Chart Creator 같은 도구를 쓰면 코드 스니펫, 구조화된 데이터, 라이브 값 같은 실제 콘텐츠를 커스텀 렌더러를 구축하지 않고도 프로덕션 품질 템플릿 안에 배치할 수 있어요.
- **락인 제로.** 오픈소스 엔진, 로컬 연산, API 키 불필요, 속도 제한 없음.

```bash
lolly qr-code --url=https://suse.com --output=og-qr.svg
lolly quotes --quote="Ship it." --output=quote.png
```

## AI 에이전트

### 모델에게 도구를 사용하라고 하세요, 환각을 일으키게 하지 말고요.

파라미터가 붙은 URL은 토큰 몇 개면 돼요. 크리에이티브 브리프에 이미지 생성까지 더하면 수천 토큰이 들고——그래도 결과물은 여전히 프레스 품질에 못 미쳐요. Lolly는 여러분의 에이전트에게 결정론적이고 검토 가능한, 바로 프로덕션에 쓸 수 있는 크리에이티브 레이어를 제공해요.


- **프로덕션 품질이 흔들리지 않아요.** 도구는 로컬에서 훨씬 적은 연산으로 프로덕션 품질의 아트워크를 만들어요. 도구는 하드코딩되어 있어요. 모델이 게을러져도 레이아웃은 그렇지 않아요. 타이포그래피 규칙, 색상 값, 여백——이것들은 프롬프트가 아니라 구조로 정해져 있어요.
- **대규모로 토큰을 절약해요.** URL 하나로 커스텀 이벤트 카드를 생성하는 비용은, 생성형 모델로 같은 결과를 프롬프트 엔지니어링해서 얻는 비용의 극히 일부예요.
- **데이터를 있어야 할 자리에 놓아요.** 구조화된 입력은 구조화된 템플릿에 매핑돼요. 발표자 이름, 세션 시간, 제품 버전——언제나 디자인이 기대하는 정확한 자리에 배치돼요.
- **결정론적이고, 감사 가능하고, 버전 관리가 돼요.** 모든 출력은 입력으로부터 재현할 수 있어요. 프로덕션 에셋에 확률적인 놀라움은 없어요.

```
Use Lolly to invite the team to KubeCon.

Parameters:
  title: "KubeCon 2026"
  date: "2026-11-10"
  location: "Atlanta"

Output the file URL.
```

## 대기업 IT 및 보안 팀

### 크리에이티브 파일이 보안 리스크가 되어서는 안 돼요.

누군가 "일단 보기 좋게 만들려고" 파일을 서드파티 서비스에 업로드할 때마다, 그건 언제든 터질 수 있는 데이터 유출 사건의 씨앗이 돼요. Lolly는 그 문제를 근원에서 없애요——크리에이티브 제작이 온디바이스, 온프레미스로, 그리고 여러분의 통제 아래에 머물러요.

- **데이터가 기기를 벗어나지 않아요.** 기본적으로 어떤 서드파티에도 아무것도 전송되지 않아요——클라우드 렌더링도, 애널리틱스도, 텔레메트리도 없어요. 동료가 만든 것은 각자의 기기에 머물러요. Strip Hidden Data 같은 온디바이스 유틸리티는 이미지, SVG, PDF에서 EXIF와 메타데이터를 브라우저 안에서 완전히 제거해요——업로드는 전혀 일어나지 않아요. PDF나 다운로드 전체, 공유 링크를 비밀번호로 잠그는 작업조차 온디바이스에서 일어나요. 비밀번호는 기기를 벗어나지 않아요.
- **완전한 에어갭 통제를 위해 셀프호스팅하세요.** 여러분 자체 인프라에 배포하세요. 서버 측 처리도, 데이터베이스도, 백엔드도 없어요. 방화벽 뒤에서 여러분만의 접근 정책으로 완전히 오프라인으로 운영할 수 있어요.
- **벤더 노출 범위를 줄여요.** 하나의 오픈소스 플랫폼이 크리에이티브 제작을 위해 난립한 SaaS 구독을 대체해요. 벤더가 적으면 계약도, 감사도, 침해 경로도 줄어들어요.
- **인프라 수준에서 브랜드 거버넌스를 강제해요.** 도구는 한 번 작성되어 데이터로 배포돼요——파일도, 수작업 프로세스도 아니에요. 밖으로 나가는 것은 정확히 승인받은 그대로예요.
- **중요한 정보를 즉시 사람이 읽을 수 있는 형식으로 만들어요.** 사고 커뮤니케이션, 컴플라이언스 보고서, 경영진 브리핑——구조화된 데이터가 디자인 병목 없이 몇 초 만에 출판 품질의 결과물이 돼요.
- **엔터프라이즈 규모를 위한 하드닝.** Lolly는 비공개 파일럿 단계의 프로토타입이에요. 암호화 및 파일 파싱 엔진은 현재 SUSE의 엄격한 인프라 강화 과정을 거치며 엔터프라이즈 규모를 준비하고 있어요——이 분야는 저희가 정말 잘하는 부분이에요. 출처 정보와 온디바이스 암호화는 설계상 이미 견고해요. 이 강화 작업이 끝나기 전까지는 범위를 한정한 맥락에서 파일럿해 보세요. [도입과 거버넌스](/info/adoption-governance.html#status)를 참고하세요.

## 제작자에 대하여

**아직 완료되지 않은 비공개 파일럿 단계의 사내 프로토타입이에요.** 완성된 제품이 아니라 빠르게 움직이는 실험이에요. 암호화 및 파싱 엔진은 현재 SUSE의 엄격한 인프라 강화 과정을 거치며 엔터프라이즈 규모를 준비하고 있어요. [Lolly가 지금 어디쯤 와 있는지 보기 →](/info/adoption-governance.html#status)

**100% 무료이고 오픈소스예요.** SaaS 요금도, 사용량 제한도, 벤더 종속도 없어요.

**어디서나 동작해요.** Web PWA, Mac, Linux, iOS, Android, CLI, 터미널 TUI——같은 엔진, 같은 결과물이에요.

**오프라인에서도 동작해요.** 로컬 연산, 로컬 저장소예요. 첫 동기화 이후에는 인터넷이 필요 없어요.

**포맷, 입력과 출력 모두.** **25**가지 내보내기 포맷을 만들고 **13**가지 원본 포맷을 가져올 수 있어요——전체 목록은 아래에서 확인하세요.

**프로덕션 품질.** 아웃라인 처리된 텍스트, 별색(Spot color) 지원, 60FPS · 스튜디오급 미디어예요.


## 도구에 대하여


디자인용 자판기라고 생각하세요. 선택하면 결과가 나와요. 매번요.

도구란 무엇을, 어떻게 만들지 정확히 아는, 바로 쓸 수 있는 크리에이티브 템플릿이에요. 제목, 날짜, 숫자를 주면 사양에 맞는 완성된 에셋을 만들어내요.

디자인 경험은 필요 없어요. 폰트, 색상, 여백에 대한 결정도 필요 없어요——도구가 이미 최선을 알고 있어요. 디자이너든, 마케터든, 개발자든, AI 에이전트든 결과물은 동일해요.

도구는 브랜드를 잘 아는 사람들이 한 번만 작성하고, 그다음부터는 다른 모든 사람이——웹 브라우저에서, 휴대폰에서, 커맨드라인에서, 또는 자동화된 파이프라인에서——사용해요. 디자인 결정은 템플릿 단계에서 고정되어 있어요. 남은 일은 콘텐츠를 채우는 것뿐이에요.
