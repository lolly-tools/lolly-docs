
# Lolly — ランディングページのコピー

数秒で本番品質のファイルが完成します——デザインルール、レイアウトロジック、書き出し品質があらかじめ組み込まれています。待つ必要も、推測する必要もありません。
お使いのデバイス上で動作します。アップロードするものも、学ぶべきこともなく、オフラインでも動作します。

---

## マーケター

### 待たずに、すべてが完成する。

引用カード、イベントタイル、ローカライズされた署名が必要なのは今日であり、来週のスプリントではありません。デザインツールを一度も開いたことがなくても、Lollyはブランドに沿った完成品を返してくれます。ルールはテンプレートの中にあるため、出力は必ず正しく仕上がります。

- **いくつかの項目を入力するだけで、完成したアセットが手に入ります。** フォントを選ぶ必要も、色を迷う必要も、「このロゴで合っているか?」と悩む必要もありません。ツールはすでに答えを知っています。あなたが用意するのは言葉だけです。
- **無限の組み合わせも、情報源はひとつ。** 引用カードを12言語にローカライズする、40のキャンペーンでブランドロックアップを差し替える、あらゆる配置向けにすべてのサイズバリエーションを生成する——デザインには一切手を加えずに実現できます。すべてはバッチグリッドのスプレッドシートから操作可能です。CSVを貼り付けるかインポートすれば、1行につき1つの完成したアセットが得られます。
- **日常的なアセット制作でクリエイティブエージェンシーがボトルネックになりません。** メール署名、イベントカウントダウン、引用カード、地図——火曜日の午後4時にチームが必要とするものすべてを、セルフサービスで用意できます。
- **コンプライアンスリスクなしに協業できます。** 顧客データがサードパーティのクラウドサービスにアップロードされることはありません。監査すべきSaaSライセンスもありません。すべてローカルデバイスの演算だけで動作します。

## セールス

### どんな商談にも、必要なものを携えて臨める。

外出先で、資料が間違っていて、顧客から特別な要望があった——そんなとき、Lollyはどんなデバイスもアセットスタジオに変えます。デザイナーも、待ち時間も、言い訳も必要ありません。

- **何も待つ必要はありません。** 外出中で、商談まで日数が残っていなくても、アセットは今すぐ必要です。今すぐ生成しましょう。
- **たった一人で、大部隊の力を。** 統率の取れたグローバルキャンペーンの力を、手のひらの上で。どの担当者も、どの地域も、同じ仕上がりの品質で。
- **世界クラスの体験。** 即座に生成されたものでも、急いで作った印象は一切ありません。一度きりのものは何もなく、やり取りを重ねるたびに、少しずつ洗練されていきます。
- **チームの速さに驚くはず!** 彼らはもう眠っています。これは、私が最初に必要になったときにすでに解決されていました。

## ジャーナリスト

### スピードと精度のために作られた、あなたの編集ツールキット。

編集スタイルを一度構築すれば、リアルタイムのデータから出版品質のアセットをその場で生成できます。

- **データを直接ビジュアルに反映します。** 構造化データをチャート、地図、テーブルのテンプレートに接続します。数値を更新するだけで、レイアウトは自動的に調整されます。
- **出版物のスタイルに正確に一致します。** ツールはタイポグラフィと配色の制約をハードコードして作成されており、ハウススタイルは緩やかなガイドラインとしてではなく、テンプレートレベルで強制されます。
- **印刷用にも画面用にも対応。** 同じテンプレートからSVG、CMYK PDF、高解像度PNG、60fps動画を書き出せます。ひとつのソースで、制作デスクが必要とするあらゆる形式に対応します。
- **繰り返し発生するニュースのために、再利用可能なフォーマットを。** 選挙結果、四半期決算、気象イベント——テンプレートを一度作成すれば、そのストーリーが発生するたびに再利用できます。

## オペレーション

### 日常のアウトプットに、産業レベルの厳密さを。

すべてがマーケティングというわけではありません。委託ラベル、バッジの一括発行、コンプライアンス用の一枚もの資料、インシデント通知——レイアウトそのものが構造を支え、データが決して誤っていてはならないアウトプットです。Lollyは構造化されたコンテンツを、卓越したデザインと同じ厳密さで扱います。

- **構造化データを入力すれば、正確な出力が得られます。** CSVやJSONのテーブルを入力すれば、すべてのフィールドがテンプレートの想定通りの位置に正確に配置されます——1行につき1つの完成ファイルです。
- **物理的な精度。** 実寸単位(mm/cm/in/pt)、正確なDPI、裁ち落としとトンボ。ラベルプリンターや印刷会社は、必要なものをそのまま受け取れます。
- **ロジックはツールの内部で実行されます。** バーコード、日付計算、条件付きレイアウト、コントラストチェック——これらはレンダリング時に計算され、手作業ではありません。
- **設計上、決定論的です。** 同じ入力からは、常に同じファイルが生成されます。CLIやパイプラインから、監査可能・再現可能・自動化可能です。

## メディア&クリエイティブ

### 機械が最も得意とすることは、機械に任せましょう。

あなたが設計するのはシステムであり、解決するのは課題であって、一点物の制作ではありません。Lollyは、あなたのデザイン上の決定を、組織全体が使えるツールへと変換する実行レイヤーです——すべてのアセット制作にあなたが関わる必要はありません。

- **これはフロンティアです。** 高度なフィルター、コンポジット、アニメーション、生成的デザイン——このプラットフォームに上限はありません。あらゆるWeb技術がテンプレート作成者に開かれています。どのクリエイティブスイートよりも多くのことを、本番品質に仕上げる力とともに実現できます。
- **ツールがツールを組み合わせます。** あるツールは、別のツールの出力をライブアセットとして埋め込めます——自身のQRコードを生成する名札、ライブチャートを組み込むカード——デザインツールも手作業でのコンポジットも不要です。
- **ファイルではなく、ツールを作りましょう。** 10,000枚のソーシャルカードを一枚ずつ作るのではなく、10,000枚を生成するテンプレートを作りましょう。あなたの時間はデザイン上の課題に使われ、量産作業には使われません。
- **面倒な作業をなくします。** 最適な組版、ロゴの配置、地図ピンの位置決め、カラーコントラストチェック、書き出しサイズ——すべてテンプレート層で解決されます。ルールは一度定義するだけで、繰り返し作業は機械に任せられます。
- **変わってはいけないものを固定し、変わるべきものは自由にします。** ブランドの制約はハードコードします。実際に変化すべき変数だけを公開します。ツールがクリエイティブのガードレールになります。

## デベロッパー

### ビルドパイプラインには、ビジュアルも含まれるべきです。

画像はビルド成果物です。そのように扱いましょう。Lollyはコマンドラインから実行できるため、他のあらゆるものと同じ方法でアセットを生成できます——再現可能に、自動的に、そしてワークフローの一部として。

- **生成AIメディアの賭けをスキップしましょう。** 生成AIメディアは試した——高価で、精度も十分ではありませんでした。Lollyは無料で、より高速な、ローカルで完結するコンテンツ自動化により、生成AI特有の粗さも、あなたの手間も取り除きます。
- **再現可能な出力。** 同じ入力からは、常に同じファイルが生成されます。URLをコミットし、必要なときに再生成——もう画像をリポジトリにコミットしたり、Slackで最新版を探し回ったりする必要はありません。
- **メディアをコードベースの外に出しましょう。** OG画像、QRコード、ソーシャルカード、データビジュアルを、Gitにバイナリを保存する代わりにビルド時に生成します。
- **アセットの内部でロジックを実行します。** Code CanvasやChart Creatorのようなツールを使えば、コードスニペット、構造化データ、ライブの値といった実際のコンテンツを、独自のレンダラーを構築することなく、本番品質のテンプレート内に配置できます。
- **ロックインなし。** オープンソースのエンジン、ローカル演算、APIキー不要、レート制限なし。

```bash
lolly qr-code --url=https://suse.com --output=og-qr.svg
lolly quotes --quote="Ship it." --output=quote.png
```

## AIエージェント

### モデルに、ハルシネーションではなくツールの利用を指示しましょう。

パラメータ付きのURLはわずか数トークンです。クリエイティブブリーフに画像生成を加えれば数千トークンにもなり——それでも結果はプレス品質には届きません。Lollyはあなたのエージェントに、決定論的でレビュー可能な、本番対応のクリエイティブレイヤーを提供します。


- **本番品質がぶれることはありません。** ツールはローカルで動作し、わずかな計算量で本番品質のアートワークを生成します。ツールはハードコードされています。モデルが手を抜いても、レイアウトは手を抜きません。タイポグラフィのルール、色の値、余白——これらは構造として組み込まれており、プロンプトで指示されたものではありません。
- **大規模にトークンを節約できます。** URLからカスタムイベントカードを生成するコストは、同じ出力を生成モデルでプロンプトエンジニアリングするコストのごく一部で済みます。
- **データを、あるべき場所に。** 構造化された入力は構造化されたテンプレートに対応します。登壇者名、セッション時間、製品バージョン——毎回、デザインが想定した通りの位置に正確に配置されます。
- **決定論的で、監査可能で、バージョン管理されています。** すべての出力はその入力から再現可能です。本番アセットに確率的な驚きはありません。

```
Use Lolly to invite the team to KubeCon.

Parameters:
  title: "KubeCon 2026"
  date: "2026-11-10"
  location: "Atlanta"

Output the file URL.
```

## 大企業のITおよびセキュリティチーム

### クリエイティブファイルが、セキュリティリスクになってはいけません。

誰かが「とにかく見た目を整えるために」ファイルをサードパーティのサービスにアップロードするたびに、それはいつか起こりうるデータ流出事件の種になります。Lollyはその問題を根本から取り除きます——クリエイティブ制作はデバイス上、オンプレミスで、あなたの管理下にとどまります。

- **データがデバイスの外に出ることはありません。** デフォルトでは、いかなるサードパーティにも何も送信されません——クラウドレンダリングも、アナリティクスも、テレメトリもありません。同僚が作成したものは、それぞれのマシン上にとどまります。Strip Hidden Dataのようなオンデバイスユーティリティは、画像、SVG、PDFからEXIFやメタデータをブラウザ内で完全に除去します——アップロードは一切発生しません。PDFやダウンロード全体、共有リンクをパスワードでロックする場合でさえ、それはデバイス上で行われます。パスワードがデバイスの外に出ることはありません。
- **完全なエアギャップ管理のためにセルフホストできます。** 自社のインフラにデプロイできます。サーバーサイド処理も、データベースも、バックエンドもありません。ファイアウォールの内側で、独自のアクセスポリシーのもと、完全にオフラインで運用できます。
- **ベンダーの露出範囲を減らします。** ひとつのオープンソースプラットフォームが、クリエイティブ制作のために乱立したSaaSサブスクリプション群を置き換えます。ベンダーが少なければ、契約も、監査も、侵害の経路も少なくなります。
- **インフラレベルでブランドガバナンスを強制します。** ツールは一度だけ作成され、データとして配布されます——ファイルとしてでも、手作業のプロセスとしてでもありません。外に出ていくものは、外に出すことが承認されたものと寸分違いません。
- **重要な情報を、瞬時に人が読める形式にします。** インシデント連絡、コンプライアンスレポート、経営層向けブリーフィング——構造化データは、デザインのボトルネックなしに、数秒で出版品質の出力になります。
- **エンタープライズ規模に向けたハードニング。** Lollyはクローズドパイロットのプロトタイプです。暗号化とファイル解析のエンジンは現在、SUSEによる厳格なインフラハードニングを経て、エンタープライズ規模に備えている最中です——私たちはこの領域を非常に得意としています。来歴管理とオンデバイス暗号化は設計上すでに堅牢です。そのハードニングが完了するまでは、範囲を限定した文脈でパイロット運用してください。[導入とガバナンス](/info/adoption-governance.html#status)をご覧ください。

## ツールについて


デザインの自動販売機だと考えてください。選択すれば、結果が出てきます。毎回、必ず。

ツールとは、何を、どのように作るかを正確に理解した、すぐに使えるクリエイティブテンプレートです。見出し、日付、数字を渡せば、仕様通りの完成したアセットが生成されます。

デザインの経験は不要です。フォント、色、余白について判断する必要もありません——ツールがすでに最適な答えを知っています。デザイナーであっても、マーケターであっても、デベロッパーであっても、AIエージェントであっても、出力は同じです。

ツールは、そのブランドを熟知した人々によって一度だけ作成され、それ以外のすべての人が——ウェブブラウザから、スマートフォンから、コマンドラインから、あるいは自動化されたパイプラインから——利用します。デザイン上の判断はテンプレートレベルで固定されています。あとはコンテンツを入力するだけです。
