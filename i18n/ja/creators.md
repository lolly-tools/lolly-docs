# クリエイターのための Lolly

良いものを、素早く、しかも正しく仕上げたい——そう思っているはずです。Lolly はまさにそのために作られています。いくつかの項目を入力するだけで、完成したファイルが手に入ります。レイアウト、書体、ルールはツールによって強制されるのであって、あなたの目分量ではありません。デザインソフトは不要、待ち時間もありません。あなた自身のデバイス上で動作し、オフラインでも使えます。

**あなたにとってのメリット:** 数分で仕上がるのに、何時間もかけたように見えます。デザイナーの手が空くのを待つ必要はもうありません。ブランドルールが組み込まれているため、ブランドから外れたアセットにあなたの名前がつくこともありません。あなたはただ、速く、洗練されていて、信頼できる存在に見えるだけです。

初めての方は、**[クイックスタート](/info/quickstart.html)** から始めてください——ブランドを取り込み、最初のファイルを書き出すところまで案内します。このページは、あなたがここに留まりたくなる理由です。

## Lolly を選ぶ理由——あなたのアドバンテージ

- **細かい調整不要、すぐ完成。** ツールはフォント、カラー、ロゴ、余白をすでに把握しています。あなたが言葉を用意すれば、アセットが返ってきます。迷う必要はありません。
- **構造的に正しい。** デザインルール、レイアウトの制約、動作するロジックはすべてテンプレートの中にあります——ツールの作者が意図的に自由度を残さない限り、出力が仕様からずれることはありません。しっかりとしたガードレールの中で、自由な創造の幅を確保します。
- **ひとつの元データから、あらゆる形式へ。** 同じデザインが SVG、印刷用 PDF、高解像度 PNG、60fps の動画として書き出せます。一度作れば、どこにでも展開できます。
- **無限のバリエーション。** カードを十数言語にローカライズしたり、配置ごとにあらゆるサイズを生成したり、40件のキャンペーンにまたがってロックアップを差し替えたり——すべて1つのデザインから、スプレッドシートで駆動できます。
- **あなたのもの、オフライン、プライベート。** 何もアップロードされません。アカウントもクラウドも、監査すべきライセンスもありません。飛行機の中でノートパソコンを開いたまま作業を続けられます。
- **オープンで、ロックインされない。** エンジンはオープンソースであり、あなたの成果物はベンダーのデータベースの中ではなく、ただの URL として存在します。

## 他ツールとの比較

Lolly は Canva や Figma を目指しているわけではありません。Lolly が力を発揮するのは、*自由な操作性よりも、ルール・再現性・厳格な制約が重要になる*場面です——デザイン上の判断をツール化することで、チーム全員が、あなたが毎回関わらなくても使えるようにします。

|---|---|---|---|---|---|
| 大量コンテンツ生成 | 一部対応 | ✗ | ✗ | ✗ | **✓** |
| 完全オフライン動作 | ✗ | ✗ | ✓ | 一部対応 | **✓** |
| テンプレートロジックと厳格な制約 | ✗ | 一部対応 | ✗ | 一部対応 | **✓** |
| デザインスキル不要 | 一部対応 | ✓ | ✗ | ✗ | **✓** |
| 自動 Content Credentials | ✗ | ✗ | 一部対応 | ✗ | **✓** |
| ツールが他のツールを組み合わせる | ✗ | ✗ | ✗ | ✗ | **✓** |
| オープンなエンジン、SaaS にロックインされない | ✗ | ✗ | ✗ | 一部対応 | **✓** |
| C2PA content credentials  | ✗ | ✗ | ✗ | ✗ | **✓** |

**Lolly を使うべきなのは**、何度も繰り返し作るアセットのときです——イベントサイネージ、名言カード、ローカライズされたバナー、メール署名、配送ラベル、SNS 用バリエーション、データ駆動のチャートや地図など。**デザインツールを使うべきなのは**、一点物のヒーロービジュアル、単発のキャンペーンコンセプト、あえてシステムから外れる*ことを意図した*アイデア出しのときです。Lolly はシステムを実行するものであり、スタジオに取って代わるものではありません。

## 作れるもの

### 待たずに、すべて正しく

いくつかの項目を入力するだけで、完成したアセットが手に入ります——フォントを選ぶ必要も、カラーに迷う必要も、「このロゴで合っているだろうか」と考える必要もありません。ツールはすでにすべて把握しています。メール署名、イベントのカウントダウン、名言カード、バッジ、地図——チームが火曜の午後4時に必要とするものすべてを、セルフサービスで用意できます。しかも顧客データが端末の外に出ることはないため、コラボレーションにコンプライアンス上のリスクはありません。

### ライブデータから完成したビジュアルへ

**CSV** や **JSON** のテーブルをチャート、地図、表テンプレートに接続し、数値を更新するだけです——レイアウトは自動で調整されます。選挙結果、四半期決算、気象イベントなど、フォーマットを一度作れば、記事が出るたびに何度でも再利用できます。ハウススタイルに*正確に*一致します。それはゆるいガイドラインとしてではなく、テンプレートレベルで強制されているからです。

### ツールがツールを組み合わせる

あるツールが、別のツールのライブ出力をアセットとして埋め込むことができます——自分自身の QR コードを描画するネームバッジや、ライブチャートを差し込んだカードなど、デザインツールも手作業での合成も不要です。埋め込まれたレンダリングはライブのまま保たれ、読み込みのたびに再描画されます。平坦化された画像になることは決してありません。

### 必要なときのためのフロンティア

高度なフィルター、合成、アニメーション、ライブカメラエフェクト、生成的デザイン——あらゆる Web 技術をテンプレート作者が利用できるため、天井は非常に高く設定されています。*ファイル*ではなく*ツール*を作りましょう。1万枚のカードを作るのではなく、1万枚のカードを生成する仕組みを作るのです。変わるべきでないものはロックし、変わるべきものだけを公開しましょう。

## 気づくはずの細部

- **印刷を正しくこなす。** 実寸単位(mm/cm/in/pt)と DPI、塗り足しとトンボ、CMYK と PDF/X-4 の出力インテント——印刷会社が必要とするものをそのまま渡せます。詳しくは [書き出し](/info/exporting.html#size-print-units) を参照してください。
- **動画とモーション。** WebM/MP4(音声トラック付き)、アニメーション PNG/WebP、アニメーション SVG、Lottie を書き出せます——静止画を作るのと同じテンプレートから。
- **編集可能なベクター。** SVG、EPS、EMF、DXF での書き出しに対応。ベクター出力ではテキストがアウトライン化されたパスになるため、印刷会社やカッティングマシンはどこでも開ける精細なジオメトリを受け取れます。(アウトライン化されたテキストは Illustrator 上で*再入力*できなくなります——フォントなしで完全な忠実度を得るためのトレードオフです。)
- **積み重なる利便性。** スプレッドシートをまとめてアセットのフォルダに一括変換したり、セッションを保存してプロジェクトに整理したり、状態を URL として共有したり、フォルダ全体を1つの zip として書き出したりできます。
- **実際に心地よい UI。** どこでも使えるアンドゥ/リドゥ、オフにできるサウンドレイヤー、テーマの切り替え、ライブカメラフィルター、メーソンリーレイアウトのギャラリー——そして、より落ち着いた低刺激なインターフェースのための **Neurospicy Mode** も。
