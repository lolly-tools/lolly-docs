# 隐私政策

*最后更新：2026年6月*

## Lolly 应用

Lolly 完全在你的浏览器中运行。**我们不收集任何数据，不传输任何数据，也没有能看到你数据的服务器。** 没有分析工具，没有追踪，也没有任何第三方。

**任何地方都不使用 cookie。** Lolly 从不设置 cookie。为了让应用正常运行，它会在**你自己的设备上**保存少量数据，全部都是你正在使用的某项功能所必需的：

- **你的浅色/深色主题**以及少量界面偏好设置（侧边栏宽度、缩放）。
- **工具目录的离线缓存**，让画廊在没有网络连接时也能加载。
- **仅保存在本地的使用计数器**，用于个人资料卡片上的小型统计数据——这些数据从不会被发送到任何地方。
- **你自己的文档和已保存的会话**，保存在浏览器本地（IndexedDB）中，让你的工作在多次访问之间得以保留。

这些数据都不会被分享、上传，也不会用于识别或追踪你，因此没有什么需要你同意——只有这份说明，让你了解保存了哪些数据。你可以随时通过**个人资料 → 清除我的所有数据**清除这一切，也可以在浏览器中清除该网站的存储数据。

这个文档站点（`/info`）更加轻量：它**不设置任何 cookie**，只在你的设备上保存浅色/深色偏好设置，并且所有内容——包括字体——都直接由 lolly.tools 自己提供，不涉及任何 CDN 或第三方请求。

## 设备端工具

有些工具是**实用工具**，作用于*你*提供的文件——例如**清除隐藏数据**，它会显示图片或 PDF 中隐藏的数据（GPS 位置、相机型号、作者、编辑者和文档元数据），并返回一份干净的副本；又如**压缩 PDF**，它会在你的设备上直接重新编码图片，从而缩小 PDF 文件体积。

这些工具**完全在你的浏览器中运行**。你选择的文件会被读入设备内存，在本地完成转换，然后作为下载文件返回给你。**它绝不会被上传**——因为根本没有可以上传的服务器。清理后的副本不带任何水印，也不含我们自己添加的任何识别性元数据；这类工具的全部意义就是*移除*数据，而不是添加数据。离开页面后不会保留任何数据，这些工具也可以离线使用。你会在每一个工具上看到一个**"在你的设备上运行——不会上传任何内容"**的徽章。

这与常见的"压缩这个 PDF"／"转换这个 HEIC"类网站完全相反，那些网站会把你的文件上传到陌生人的服务器上，去做你的浏览器本就能在本地完成的工作。

## 浏览器扩展

**Lolly URL 截图**浏览器扩展不收集、不存储、也不传输任何个人数据。没有分析工具，没有追踪，也没有远程服务器。

## 它的功能

当你要求 Lolly 网页应用（[lolly.tools](https://lolly.tools)）对某个 URL 进行截图时，该扩展会在一个临时的后台标签页中打开该页面，使用 DevTools Protocol 在你的浏览器中完成截图，将图片返回给应用，然后关闭标签页。整个过程都在你自己的设备和网络上本地完成。

## 数据

- **我们不收集任何数据。** 该扩展没有自己的服务器，也不会发起任何网络请求。
- **捕获的图片**会直接发送到同一浏览器中的 Lolly 应用——扩展本身从不上传它们。
- **你截图所用的 URL** 仅用于加载那一次截图所对应的那一个页面，不会被记录或分享。

## 权限

- **`debugger`**——通过 DevTools Protocol 捕获渲染后的页面（与 Lolly 桌面应用使用的机制相同）。
- **标签页访问权限**——用于打开和关闭加载页面的临时标签页。
- **主机访问权限**——因为你选择截图的页面可能位于任何网站上。

这些权限都不会被用来读取、监控或传输你的浏览记录。
